## Git

Git হলো Distributed Version Control System (DVCS)।
Source Code-এর পরিবর্তনের (Version) History সংরক্ষণ করে।
Offline-এ কাজ করা যায়।
কে, কখন, কী পরিবর্তন করেছে তা ট্র্যাক করা যায়।

## Common Git Commands

git init          # Repository তৈরি
git status        # বর্তমান অবস্থা দেখা
git add .         # সব File Stage করা
git commit -m "message"   # Commit করা
git log           # Commit History দেখা
git branch        # Branch দেখা
git switch -c feature     # নতুন Branch তৈরি
git merge feature         # Branch Merge
git pull          # Remote থেকে Update আনা
git push          # GitHub-এ Code পাঠান

## GitHub

GitHub = Online Repository + Collaboration
GitHub হলো Git Repository Hosting Platform।
Code Online-এ সংরক্ষণ করে।
Team Collaboration সহজ করে।
Pull Request, Code Review এবং CI/CD (GitHub Actions) সমর্থন করে।
