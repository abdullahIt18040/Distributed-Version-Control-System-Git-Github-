## Brach
```
Git Branch হলো একটি আলাদা workspace বা development line,
যেখানে আপনি নতুন feature তৈরি করতে, bug fix করতে বা experiment করতে পারেন মূল project (main branch)-কে প্রভাবিত না করে।
 যখন কাজ শেষ হয় এবং সবকিছু ঠিক থাকে, তখন সেই branch-এর পরিবর্তন merge করে main branch-এ আনা হয়।
"A Git branch is a separate workspace where you can develop new features, fix bugs,
 or experiment without affecting the main project. The changes only affect the main branch after they are merged."
```
